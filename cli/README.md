# Scrappy cli

Command line tool for Scrappy Life.

# Requirement

      composer install

# Run

      bin/scrappy posts --help

## Voir les posts

      bin/scrappy posts 200 | jq

## Charger les posts dans MongoDB

      bin/scrappy load 200 -vv
